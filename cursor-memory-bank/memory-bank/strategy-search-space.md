# Пространство поиска стратегий

## Исходные параметры

| Параметр | Значение | Описание |
|----------|----------|----------|
| I | 22 | Количество доступных индикаторов |
| C | 6 | Количество типов условий |
| S | 4 | Количество типов стопов |
| P | 250,000 | Количество комбинаций параметров оптимизации |
| i | 4 | Индикаторов в стратегии |
| c | 4 | Условий в стратегии |
| s | 1 | Стопов в стратегии |

## Формула расчёта

### Структурные комбинации

```
N_структур = I^i × C^c × S^s
```

Где:
- `I^i` — комбинации индикаторов (выбор с повторениями)
- `C^c` — комбинации условий (выбор с повторениями)
- `S^s` — комбинации стопов

### Полное пространство поиска

```
N_полное = N_структур × P
```

## Расчёт для текущей конфигурации

### Шаг 1: Индикаторы
```
22^4 = 234,256
```

### Шаг 2: Условия
```
6^4 = 1,296
```

### Шаг 3: Стопы
```
4^1 = 4
```

### Шаг 4: Структурные комбинации
```
234,256 × 1,296 × 4 = 1,214,234,624 ≈ 1.2 млрд
```

### Шаг 5: С параметрами оптимизации
```
1,214,234,624 × 250,000 = 303,558,656,000,000,000 ≈ 3 × 10^17
```

## Эффективность генетического алгоритма

При популяции 100 особей и 1000 поколений:

```
Исследовано точек = 100 × 1000 = 100,000
Процент от пространства = 100,000 / (3 × 10^17) = 3.3 × 10^-11 %
```

ГА исследует **0.000000000033%** пространства, но благодаря эволюционному отбору находит хорошие решения.

## Общая формула

```
N = I^i × C^c × S^s × P

Где:
  I = количество индикаторов в пуле
  i = индикаторов в стратегии
  C = типов условий
  c = условий в стратегии
  S = типов стопов
  s = стопов в стратегии
  P = комбинаций параметров
```
